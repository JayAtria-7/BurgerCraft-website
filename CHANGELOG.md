# Changelog

All notable changes to the BurgerCraft project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-08

### Added
- Initial release of BurgerCraft website
- Modern hero section with parallax scrolling effect
- Animated about section with fade-in effects
- Interactive menu showcase with hover animations
- Reservation form with smooth transitions
- Customer testimonials section
- Image gallery with zoom effects
- Contact section with form validation
- Custom footer with social media links
- Sticky navigation header
- Smooth scrolling between sections
- Ripple effect on buttons
- Loading animations
- Intersection Observer for scroll-triggered animations
- Responsive design for all screen sizes
- Glass morphism effects
- Multiple keyframe animations (fadeInUp, fadeInLeft, fadeInRight, scaleIn, float, pulse, slideDown)
- Font Awesome icon integration
- Google Fonts integration (Noto Sans, Playfair Display)

### Features
- Pure HTML, CSS, and JavaScript (no frameworks)
- Mobile-first responsive design
- Cross-browser compatible
- SEO-friendly structure
- Accessibility considerations
- Fast loading performance
- Smooth 60fps animations

### Styling
- Modern gradient backgrounds
- Custom color palette
- Professional typography
- Enhanced shadows and depth
- Hover effects on all interactive elements
- Form styling with focus states
- Image overlay effects
- Card-based layouts

### JavaScript Features
- Smooth scroll navigation
- Intersection Observer API usage
- Form submission handling
- Parallax effect implementation
- Dynamic header behavior
- Ripple button effects
- Loading transitions
- Cursor trail effect (desktop only)
